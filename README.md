# AuviaMart - Premium E-Commerce Platform

AuviaMart is a sophisticated, full-stack e-commerce platform designed with a "Phone-First" responsive strategy and a premium aesthetic. It features a deep teal and gold palette, fluid animations, and a curated shopping experience.

## ✨ Features

- **Premium Design:** Deep teal and gold color palette with sophisticated typography (`Playfair Display` and `Inter`).
- **Responsive & Phone-First:** Optimized for mobile users while providing a rich desktop experience.
- **Fluid Animations:** Powered by `Framer Motion` and `GSAP` for smooth transitions and micro-interactions.
- **Product Catalog:** High-resolution product listing and detailed single product pages.
- **Admin Dashboard:** Comprehensive dashboard for managing products and viewing key metrics using `Recharts`.
- **Authentication:** Integrated Sign-in and Sign-up flows.
- **State Management:** Efficient cart and wishlist management using `Zustand`.
- **Theme Support:** Native light and dark mode support with `next-themes`.
- **Interactive UI:** WhatsApp integration, announcement tickers, and high-end hero carousels.

## 🚀 Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **State Management:** [Zustand](https://zustand-demo.pmnd.rs/)
- **Charts:** [Recharts](https://recharts.org/)

## 🛠️ Getting Started

### Prerequisites

- Node.js 20+
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd Roshaan
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable React components (UI, Layout, Home, Products).
- `src/lib`: Utility functions and data definitions.
- `src/store`: Zustand state management stores.
- `public`: Static assets including images and logos.

## 🌐 Deployment

This project is optimized for deployment on [Vercel](https://vercel.com/).

```bash
npm run build
```
